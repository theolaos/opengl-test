Learning OpenGL with C++

## Running the project

- Linux:

    ```bash
    $ git clone https://github.com/theolaos/opengl-test
    $ cd opengl-test
    ```
    Then create the build folder:
    ```bash
    $ mkdir build/
    $ cd build
    ```
    Then finally to compile type:
    ```bash
    $ cmake ..
    $ make
    ```
